## كيف تستخدم هذا المستودع؟

1. اختر تخصصك من الجدول أعلاه
2. افتح مجلد التخصص واقرأ `README.md` أولًا
3. اتبع الملفات بالترتيب الرقمي
4. كل ملف يحتوي على: مفاهيم + أمثلة عملية + مصادر + وقت مقدر
