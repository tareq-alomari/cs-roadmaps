## الشهادات الأهم بالتفصيل

### CompTIA Security+ — للبداية

```
السعر:    ~$400
الصعوبة:  ⭐⭐☆☆☆
الوقت:    2-3 أشهر دراسة

المحتوى:
- Threats, Attacks & Vulnerabilities
- Architecture & Design
- Implementation
- Operations & Incident Response
- Governance, Risk & Compliance

مناسب لـ: الدخول لسوق العمل

مصادر الدراسة:
- Professor Messer (مجاني على YouTube)
- CompTIA CertMaster
- Jason Dion على Udemy
```

### OSCP — المعيار الذهبي للـ Pentesting

```
السعر:    $1499 (90 يوم Lab)
الصعوبة:  ⭐⭐⭐⭐⭐
الوقت:    3-6 أشهر تحضير

الامتحان:
- 24 ساعة متواصلة
- 5 أجهزة تحتاج اختراقها
- تقرير مكتوب بعدها (24 ساعة)

مطلوب قبله:
✓ TryHackMe - Jr Penetration Tester path
✓ HackTheBox - 30-50 جهاز على الأقل
✓ TCM Security PNPT

نصيحة: حل أجهزة OSCP في Proving Grounds
```

### PNPT — الأفضل قيمة مقابل السعر

```
السعر:    $400
الصعوبة:  ⭐⭐⭐☆☆
الوقت:    2-3 أشهر

من TCM Security (Heath Adams)
- امتحان عملي 5 أيام
- تقرير احترافي
- يُقبل من قبل العديد من الشركات

مثالي كخطوة قبل OSCP
```

## خطة الشهادات الموصى بها للمهندس البرمجي

```
السنة الأولى:
CompTIA Security+ → eJPT → PNPT

السنة الثانية:
OSCP + شهادة سحابة (AWS Security)

السنة الثالثة حسب التخصص:
Red Team: OSEP, CRTO
Malware:  GREM (GIAC Reverse Engineering Malware)
DFIR:     GCFE, GCFA, FOR508
AppSec:   OSWE, GWEB
```

## نصائح للنجاح في الامتحانات

```
1. التطبيق أولًا — لا تعتمد على الحفظ
2. ابنِ Lab منزلي (VMs)
3. حل CTFs باستمرار (HackTheBox, TryHackMe)
4. اكتب WriteUps لما تحله
5. انضم لمجتمعات: Discord, Reddit r/netsec
6. لا تشتري شهادات بدون خبرة حقيقية
```
