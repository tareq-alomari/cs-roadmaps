# 05 — الشهادات (Mobile Development)

| الشهادة | المزود | المستوى |
|---------|--------|---------|
| Associate Android Developer | Google | مبتدئ |
| Google Developer Expert (GDE) | Google | خبير (بالترشيح) |
| Apple Certified Developer | Apple | متوسط |
| Meta React Native Certificate | Meta (Coursera) | مبتدئ |

## ملاحظة

شهادات الموبايل أقل أهمية من Portfolio قوي يضم تطبيقات منشورة فعلًا على المتجرين.

## خطة موصى بها

```
1. ابنِ 3 تطبيقات حقيقية وانشرها
2. Associate Android Developer أو React Native Meta Certificate
3. Portfolio > شهادات في هذا المجال
```
