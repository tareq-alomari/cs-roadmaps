# 06 — الشهادات (Networking)

## خارطة شهادات Cisco

```
Entry:        CCT
Associate:    CCNA (الأهم للبداية)
Professional: CCNP (Enterprise, Security, Data Center, SP)
Expert:       CCIE (الأصعب والأعلى قيمة في الصناعة)
```

## الشهادات التفصيلية

| الشهادة | المزود | المستوى | السعر |
|---------|--------|---------|-------|
| **CCNA** | Cisco | مبتدئ → متوسط | $330 |
| **CCNP Enterprise** | Cisco | متقدم | $400 × 2 |
| **CCIE Enterprise** | Cisco | خبير | $1600 |
| CompTIA Network+ | CompTIA | مبتدئ | $370 |
| Juniper JNCIA | Juniper | مبتدئ | $200 |
| **CWNA** | CWNP | Wireless متوسط | $275 |

## خطة موصى بها

```
السنة 1: CompTIA Network+ → CCNA
السنة 2: CCNP Enterprise أو CCNP Security
السنة 3+: CCIE (اختياري، للخبراء)
```

## نصيحة مهمة

```
CCNA وحده كافٍ لأول وظيفة.
ركز على الـ Lab العملي بـ Cisco Packet Tracer أو GNS3.
```
